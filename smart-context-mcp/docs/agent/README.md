# AI Agent Docs

This directory is written **for AI agents / tool integrations**.

## Start here

- `AGENT_PLAYBOOK.md` — recommended workflows using the Six Pillars
- `TOOL_REFERENCE.md` — pillar API reference (inputs + guidance)
- `ARCHITECTURE.md` — internal architecture (how pillars are orchestrated)
- `../adr/ADR-033-Six-Pillars-Architecture.md` — canonical spec for the pillars

## Six Pillars (cheat sheet)

- `navigate` → find symbols/files
- `read` → skeleton/fragment/full
- `understand` → explain structure/impact (enable `include` when needed)
- `change` → dry-run plan → apply safely
- `write` → create/scaffold files
- `manage` → status/undo/redo/reindex/history
